# 学习笔记

这个目录存放学习过程中的延伸思考、主动提问和探索记录。

## 使用方式

每次学习某篇文章时产生的疑问和延伸，创建一个对应的笔记文件。

## 命名规范

```
{日期}-{来源文件slug}-notes.md

示例：
2026-06-01-transformer-notes.md
2026-06-02-agent-memory-notes.md
2026-06-03-mcp-protocol-notes.md
```

## 笔记模板

```markdown
# {主题} 学习笔记

来源：[原文标题](../02-理论基础/03-transformer.md)
日期：2026-06-01

## 延伸问题

### Q1: 你的问题
> 回答/思考/查到的资料

### Q2: 另一个问题
> 回答

## 关键洞察

- 学到的重要点
- 和其他知识的关联

## 待深入

- [ ] 还没搞懂的点
- [ ] 想进一步研究的方向
```

## 为什么不直接写在原文里

- 原文是"知识库"，保持客观、稳定、可复用
- 笔记是"个人理解"，带有时间性和主观性
- 分开后原文可以持续更新而不影响你的笔记
- 笔记通过链接关联原文，Obsidian 的反向链接能自动建立双向关系
